GitHub 연동 (토큰 불필요)
아래 버튼을 누르면 현재 종목 데이터가 holdings.json 파일로 다운로드됩니다. ISA 모니터링 저장소 페이지에 이 파일을 끌어다 놓으면(드래그 앤 드롭) 기존 파일이 자동으로 교체되며, 별도 토큰이나 로그인 절차 없이 그대로 커밋됩니다.
순서: 다운로드 → GitHub 저장소 페이지 열기 → 파일 목록 위에 다운로드한 파일을 드래그 → "Commit changes" 버튼 클릭.
다른 기기·다른 저장소에서 쓰던 holdings.json 파일이 있다면 여기로 불러올 수 있습니다. 불러오면 현재 목록 전체가 파일 내용으로 교체됩니다.
폰 ↔ PC 자동 동기화 (비공개 저장소)
GitHub 저장소의 holdings.json을 토큰으로 직접 읽어옵니다. PC에서 다운로드→커밋한 뒤, 폰에서는 이 카드의 버튼 한 번으로 최신 상태를 바로 가져올 수 있습니다. 토큰은 이 기기의 브라우저에만 저장되며 외부로 전송되지 않습니다.
이 앱을 공개 페이지로 배포할 계획이라면 이 기능은 사용하지 마세요 — 토큰이 저장된 브라우저의 개발자도구를 통해 노출될 수 있습니다.
